# CODSOFT Machine Learning Internship

This repository contains all tasks completed for the CODSOFT ML Internship.

##  Tasks Completed

### 1. Spam SMS Detection
Classifies SMS messages as Spam or Ham using NLP & Naive Bayes.

### 2. Customer Churn Prediction
Predicts whether a customer will leave the service using classification models.

### 3. Movie Genre Classification
Predicts the genre of the movie based on its plot description.
