# Customer Churn Prediction
This project predicts whether a customer will churn (leave the bank) using historical customer data.

## Dataset
Churn Modelling Dataset (Bank Customers)

## Tools Used
1. Python

2. Pandas

3. NumPy

4. Scikit-learn

5. Matplotlib

6. Seaborn

## Features Used
1. Credit Score

2. Geography

3. Gender

4. Age

5. Tenure

6. Balance

7. Number of Products

8. Credit Card Status

9. Active Membership

10. Estimated Salary

## Algorithm
1. Logistic Regression

## Evaluation
 Accuracy achieved: 79%

Confusion Matrix used for evaluation
